# 9M2PJU QSO Map Tool

A modern, high-performance utility for amateur radio operators to visualize their logbook data on an interactive map.

![Screenshot](/img/screenshot.png)

*   **Light Tech Theme**: A modern, high-tech light mode with Tech Blue accents and Glassmorphism UI.
*   **Data Support**: Load ADIF, Cabrillo & SOTA CSV files.
*   **Advanced Visualizations**:
    *   Heatmaps (Global & Per-band)
    *   Geodesic lines (Great Circle paths)
    *   Maidenhead Grid Squares & Fields
    *   ITU & CQ Zones
    *   Worked All Britain/Ireland grids
*   **Offline First**: All processing happens client-side in your browser. No data needs to be uploaded.
*   **Mobile Ready**: Fully responsive design for use on smartphones and tablets.
*   **Privacy Focused**: No cookies, no tracking, no ads.
